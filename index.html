Privacy Policy

Last updated: April 2026

This application is used for automated Facebook Page content publishing. It does not collect, store, or share any personal data from users.

Data We Access

This app accesses Facebook Page data solely to publish video content to a Facebook Page owned and operated by the app owner. No user data is collected from third parties.

How We Use Data

Access tokens are used only to post content to the designated Facebook Page. They are stored privately and never shared with any third party.

Data Retention

No personal data is stored. Access tokens are held only in the app owner's private automation environment.

Third Party Services

This app uses the Facebook Graph API to publish content. Please refer to Facebook's Privacy Policy for information about how Facebook handles data.

Contact

For any questions about this privacy policy, please contact the page owner through the Facebook Page directly.
